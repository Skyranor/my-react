# Рекомендуемые команды (Suggested Commands)

Ниже приведены основные команды для работы над проектом `my-react`.

## 🛠️ Запуск (Development)
- **`npm run dev`** — Запуск локального сервера разработки (Vite).
  - Позволяет смотреть отображение кастомного реакта в браузере.
  - Содержит панель [Fiber Inspector](docs/INSPECTOR.md) для отладки.

## 🧪 Тестирование (Testing)
- **`npm run test`** — Запуск всех тестов проекта через `Vitest`.
  - Позволяет быстро верифицировать изменения в `react-reconciler` и `react-dom`.
- **`npx vitest <path>`** — Запуск конкретного тестового файла.

## 🔨 Сборка / Компиляция (Compilation)
- Проект использует **TypeScript** (`tsconfig.json` настроен на strict).
- Ошибки компиляции подсвечиваются в процессе разработки.
