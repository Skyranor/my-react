# Действия после завершения задачи (Completed Task Actions)

При завершении доработки или исправлении багов:

1.  **Запустите тесты**:
    - Запустите команду `npm run test`, чтобы убедиться, что тесты завершились без ошибок.
2.  **Проверьте DevTools (Inspector)**:
    - Если доработка касается рендеринга, откройте инспектор (DevTools) в браузере (через `npm run dev`) и проверьте отображение счетчиков рендеров (`x2`) и пропусков (`Skip`).
3.  **Убедитесь в отсутствии ошибок компиляции**:
    - Ошибки TS должны отсутствовать для затронутых файлов.
